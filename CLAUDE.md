## Project Overview
Deep learning-based wafer defect prediction system for semiconductor manufacturing (ASML).
Uses a U-Net architecture to predict surface topography from masked wafer images.

## Tech Stack
- **Language:** Python 3.13 (requires >= 3.10)
- **Package Manager:** UV (lock file: `uv.lock`)
- **DL Framework:** PyTorch + PyTorch Lightning
- **Web App:** Dash (interactive labeling tool)
- **Visualization:** Plotly (interactive), Matplotlib (static)
- **Data:** Polars (primary), Pandas, PyArrow (parquet files)
- **Proprietary:** `fastlibrary` (ASML), `zygorw` (Zygo file reader)
- **Build System:** Hatchling

## Domain Context
- **Wafer sides:** ZA (right 40%: 0.60–1.00) and ZE (left 40%: 0.00–0.40)
- **Process steps:** IBF → coating → HI bonding → Z polishing
- **Data format:** Zygo `.wrk` files (surface topography), parquet (metadata)
- **DB source:** `MiQaT_Specification_Data.parquet`
- **Model input:** 2 channels (masked_image + binary_mask) → 1 channel output
- **Training:** 256×512 resolution, L1 + SSIM loss, batch 8, lr 1e-4
